AI คืออะไร
และทำงานยังไง

ปูพื้นฐาน AI, ML, LLM สำหรับ Engineer ที่ไม่เคยใช้มาก่อน — ไม่ต้องมีความรู้ IT พร้อม top 10 tools ที่ใช้ในงานโรงงานได้ทันที

เป้าหมายของ Module นี้ — เข้าใจว่า AI คืออะไร ทำงานยังไง และเลือก tool ที่เหมาะกับงานแต่ละประเภทในโรงงานได้

AI, ML, LLM — อธิบายแบบ Engineer

คำว่า AI ถูกใช้กว้างมาก จนบางครั้งสับสนได้ง่าย ลองคิดแบบนี้:

Level 1

AI

Artificial Intelligence — ระบบที่เลียนแบบความฉลาดมนุษย์ ตัดสินใจ วางแผน แก้ปัญหาได้

Level 2

ML

Machine Learning — AI ที่เรียนรู้จากข้อมูล ไม่ต้องโปรแกรมทุกกฎ เพิ่มความแม่นยำได้เอง

Level 3

LLM

Large Language Model — AI ที่เข้าใจและสร้างภาษา เช่น ChatGPT, Claude, Gemini

GenAI = Generative AI — LLM ที่ generate text, code, image ได้ ChatGPT, Claude, Copilot คือ GenAI ทั้งหมด นี่คือสิ่งที่เราจะใช้ในคอร์สนี้

AI ทำงานยังไง — ทำไมมันถึงฉลาด (และพลาดได้)

LLM ถูก train จาก text จำนวนมหาศาล — หนังสือ, เว็บ, บทความ, code — จนเรียนรู้ pattern ของภาษา มันไม่ได้ "คิด" แบบมนุษย์ แต่ทำนายว่า "คำหรือประโยคต่อไปควรเป็นอะไร" ตาม context ที่ให้ไป

ด้วยเหตุนี้ AI จึง:

  • ตอบได้ดีมากเมื่อ context ชัดเจน — เหมือนมีผู้เชี่ยวชาญคอยช่วยตลอดเวลา
  • อาจผิดพลาดเมื่อข้อมูลไม่มีใน training data หรือ context ไม่ครบ
  • สร้าง "hallucination" ได้ — ตอบมั่นใจแต่ข้อมูลผิด ต้องตรวจสอบเสมอ
สำคัญมาก: AI คือ "first draft" ไม่ใช่ final answer — Engineer ต้องตรวจสอบตัวเลข, spec, และข้อเท็จจริงก่อนนำไปใช้งานจริงเสมอ

Top 10 AI Tools สำหรับ Engineer

เลือก tool ให้ตรงกับงาน — ไม่จำเป็นต้องใช้ทุกตัว:

General
ChatGPT
ถาม-ตอบ, brainstorm, เขียนรายงาน — เริ่มต้นง่าย
General
Claude
เอกสารยาว, วิเคราะห์ข้อมูล, ปลอดภัยกว่า
Google
Gemini
ผูกกับ Google Workspace ได้ — สะดวกถ้าใช้ Google
Microsoft
Copilot
ทำงานกับ Word, Excel, Teams ได้เลย — ดีที่สุดสำหรับโรงงานที่ใช้ M365
Code
Deepseek
เขียน/แก้ code, Python, VBA — ฟรีและแม่นยำสูง
Search
Perplexity
ค้นหาพร้อม citation — ตรวจสอบข้อมูลวิศวกรรม
Research
Scispace
สรุปงานวิจัย academic paper สำหรับงาน R&D
Visual
Napkin.ai
แปลง idea เป็น diagram / flowchart อัตโนมัติ
Writing
Grammarly
ตรวจ grammar, ปรับ tone เอกสารภาษาอังกฤษ
Summary
Monica
สรุปบทความ, แปลภาษาจากเอกสาร
แนะนำสำหรับโรงงาน Fiber Cement: เริ่มจาก Copilot (ถ้ามี M365) หรือ ChatGPT / Claude สำหรับงานทั่วไป Deepseek ถ้าต้องการเขียน VBA หรือ Python ฟรี

Workshop: ทดลองใช้ AI ครั้งแรก

ลองถาม AI ด้วยประโยคต่อไปนี้ แล้วสังเกตผลลัพธ์:

  • "อธิบายหลักการทำงานของ Ball Mill สำหรับการบด Clinker ให้ฉันเข้าใจแบบง่าย ๆ"
  • "ค่า Blaine Fineness คืออะไร และทำไมถึงสำคัญในการผลิต Fiber Cement"
  • "ช่วยแปลประโยคนี้จากภาษาอังกฤษเป็นภาษาไทย: [ใส่ข้อความจากคู่มือ equipment]"

สังเกตว่า AI ตอบแบบไหน ตอบถูกหรือผิด และต้องปรับ prompt อย่างไรเพื่อให้ได้คำตอบที่ดีขึ้น — นี่คือ core skill ที่จะเรียนต่อใน M02
