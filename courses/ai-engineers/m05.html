Role-based AI Application

Deep Prompt Set แยกตามแผนก — Process, QC, Maintenance, Planning, Manager พร้อม templates ที่ copy ไปใช้ได้ทันที

เป้าหมาย M05 — มี Prompt template ที่ใช้งานจริงสำหรับแผนกของตัวเอง พร้อม Deep Prompt Set ที่ทำงานได้ทันที

Process Engineer

Deep Prompt: Slurry Mixing System Design (RISE)
คุณคือ Process Engineer ในโรงงาน Fiber Cement
ปัจจุบัน Slurry Mixing มีปัญหาแยกชั้น น้ำกับปูนไม่เข้ากัน ทำให้แผ่นบางไม่สม่ำเสมอ
ช่วยออกแบบระบบ Mixing ที่ทำงานต่อเนื่อง 24 ชม. โดยลดการตกตะกอน
ข้อมูล: ถัง 2 ใบ, เครื่องคน 2 ใบพัด, flow rate 3 m³/hr
ต้องรองรับการเติม pulp และ silica slurry ด้วย
แนะนำทั้ง layout, ความเร็วรอบ, และ logic การควบคุม
ถามต่อ: ช่วยสรุปรายการอุปกรณ์ที่ต้องใช้ พร้อม model ที่มีขายในไทย
Kiln Heat Balance Analysis (CARE)
Context: Kiln heat consumption เพิ่มขึ้น 8% เทียบกับเดือนที่แล้ว
Action: วิเคราะห์สาเหตุที่เป็นไปได้ทั้งหมด
Result: ระบุ top 3 สาเหตุ พร้อม % contribution โดยประมาณ
Example: brick condition, feed rate, false air, fuel quality
Output: ตาราง สาเหตุ / likelihood / วิธีตรวจสอบ / corrective action

Quality Control Engineer

Blaine Analysis + Root Cause (CARE)
คุณคือ QC Engineer ที่เชี่ยวชาญ Fiber Cement
Context: ค่า Blaine ลดลงจาก 4,200 เป็น 3,800 cm²/g กะเช้า
Action: วิเคราะห์สาเหตุที่เป็นไปได้
Result: top 3 สาเหตุ พร้อม priority
Example: Mill liner wear, media charge ลด, feed rate สูงเกิน
Output เป็นตาราง: สาเหตุ / ความน่าจะเป็น / วิธีตรวจ / วิธีแก้ไข
ร่าง Non-Conformance Report (RTF)
คุณคือ QC Engineer
ร่าง Non-Conformance Report (NCR) จากข้อมูลต่อไปนี้:
- Batch: [ระบุ batch number]
- Defect: [ระบุ defect ที่พบ]
- จำนวน: [ระบุจำนวน/เปอร์เซ็นต์]
- วันที่: [วัน/เดือน/ปี]
รูปแบบ: เลขที่ NCR / วันที่ / คำอธิบาย / root cause / immediate action / ผู้รับผิดชอบ / due date
ใช้ภาษาทางการ สำหรับส่งให้ผู้บริหาร

Maintenance Engineer

Predictive Maintenance: สายพานเครื่องตัด (RISE)
คุณคือ Maintenance Engineer
ออกแบบระบบ Predictive Maintenance สำหรับสายพานเครื่องตัดแผ่น
โดยใช้ข้อมูล vibration + RPM + Temperature จาก sensor
เสนอ: จำนวน sensor / วิธีวิเคราะห์ + threshold / flow แจ้งเตือน PM
ต้องเชื่อมต่อกับ Excel / LINE Notify / Power Automate ได้
ถามต่อ: รายการอุปกรณ์ที่ต้องซื้อ (ยี่ห้อ, รุ่น, ราคาประมาณ)
ออกแบบระบบ LOTO (RISE)
คุณคือ Safety Engineer
ออกแบบระบบ Lockout/Tagout (LOTO) สำหรับเครื่องตัดแผ่นขนาดใหญ่
ต้องมี: ป้าย, กล่อง lock, master key, checklist รายวัน
Flow: shutdown → disconnect → lock → tag → verify
ระบุอุปกรณ์ที่ใช้ พร้อมยี่ห้อ/รุ่นที่หาได้ในไทย
ถามต่อ: ออกแบบ SOP Word format + โปสเตอร์ A4 ติดหน้างาน

Planning Engineer

สร้างแผนผลิต 14 วัน (CARE)
คุณคือ Production Planner
คำสั่งผลิต: 80,000 แผ่น (5 สูตร) ภายใน 14 วัน
เงื่อนไข: พักเครื่อง 1 วัน, ลดการเปลี่ยนสูตรให้น้อยที่สุด
ส่งทันตาม order วันที่ 10, 12, 14
Output: ตาราง 14 วัน + KPI daily + ลำดับการเปลี่ยนสูตรที่เหมาะสม
ถามต่อ: รายการวัตถุดิบที่ต้องใช้ตามแผน + Excel template

Manager / Supervisor

Executive Weekly KPI Brief (CARE)
สรุป KPI ประจำสัปดาห์จากข้อมูลต่อไปนี้:
[วาง KPI data หรือแนบไฟล์ Excel]
Output 3 bullet points:
1. KPI ที่ผ่าน target — highlight สิ่งที่ดี
2. KPI ที่ต่ำกว่า target — ระบุ % deviation และสาเหตุเบื้องต้น
3. Action items สำคัญสัปดาห์หน้า
ใช้ภาษาทางการ สำหรับนำเสนอ board meeting
