Engineering Series · EP.02

Build CPK Dashboard in 5 Minutes

ตอนนี้พาไปใช้ AI สร้างเครื่องมือจริงสำหรับงาน process capability analysis โดยใช้ prompt ช่วยออกแบบ Excel template, histogram และ dashboard ที่พร้อมใช้งานต่อกับ process จริง

The point of this episode is not just “let AI build a file for you” — but to show how an engineer can use AI to shorten setup time and focus on analysis.

โจทย์ของตอนนี้คืออะไร

หลายครั้งเวลาจะทำ dashboard ด้าน Cpk/Cp เราต้องเริ่มจากการจัดโครงสร้างไฟล์ เขียนสูตร ทำตาราง และสร้างกราฟซ้ำ ๆ ทั้งที่สิ่งสำคัญจริง ๆ คือการตีความผลลัพธ์และใช้ข้อมูลเพื่อคุยกับ process ตอนนี้จึงพาไปดูว่า AI จะช่วยย่นเวลาช่วง setup เหล่านี้ได้ยังไง

สิ่งที่ได้จาก EP.02

  • Prompt สำหรับสั่ง AI สร้าง Excel template ที่มี logic ด้าน Cp/Cpk
  • แนวทางทำ histogram และ dashboard ให้ดูง่ายขึ้น
  • โครงคิดในการ reuse template กับหลาย process
ตัวอย่าง prompt:
"ช่วยออกแบบ Excel template สำหรับคำนวณ Cp และ Cpk
จากข้อมูล measurement 1 คอลัมน์
โดยให้มี dashboard summary, histogram และส่วนแสดง USL / LSL
พร้อมอธิบายว่าต้องใช้สูตรอะไรในแต่ละช่อง"

Engineer mindset ที่สำคัญในตอนนี้

AI ไม่ได้มาแทน Excel และไม่ได้มาแทนความเข้าใจ process แต่มันช่วยให้วิศวกรสร้าง “เครื่องมือเริ่มต้น” ได้เร็วขึ้น จากนั้นเราค่อยใช้เวลาไปกับสิ่งที่มีค่ากว่า เช่น

  • ตีความว่าค่า Cpk สะท้อน process อย่างไร
  • ดูว่าความแปรปรวนมาจากอะไร
  • คุยต่อกับ production หรือ quality ว่าควรปรับตรงไหน

ใช้ต่อยังไงกับงานจริง

วิธีที่ได้ผลที่สุดคือไม่ใช่แค่ดูคลิป แต่ลองเอา prompt ไปปรับให้เข้ากับ process ของตัวเอง ใส่ data จริง แล้วดูว่า dashboard ที่ได้ตอบคำถามหน้างานหรือไม่ ถ้ายังไม่ตอบ ก็ค่อย iterate prompt ต่อจนกว่าจะได้หน้าตาที่ใช้งานจริง
