"ทุกครั้งที่เปลี่ยน process ต้องนั่งสร้าง Excel ใหม่ เขียนสูตร Cp/Cpk ใหม่ วาด histogram ใหม่... EP นี้มาแก้ตรงนั้นเลยค่ะ"

ใครทำงาน QC หรือ Process Engineer น่าจะเคยเจอปัญหานี้กันค่ะ แทนที่จะโฟกัสการวิเคราะห์ข้อมูล กลับเสียเวลา setup template ซ้ำๆ ทุกครั้งที่เปลี่ยน process

EP.02 นี้ลองใช้ AI สร้าง CPK Dashboard พร้อมใช้งานจริงใน 5 นาที มาดูกันว่าทำได้จริงไหมค่ะ 🌿

✦
Section 01

สิ่งที่ AI สร้างให้ใน EP นี้

  • 1
    Excel Template คำนวณ Cp/Cpk อัตโนมัติ ใส่ USL, LSL และ data เข้าไป สูตรคำนวณ Cp, Cpk พร้อมแสดงผลทันที ไม่ต้องเขียนใหม่ทุกครั้งค่ะ
  • 2
    Histogram + Dashboard อัตโนมัติ กราฟแสดงการกระจายของ data พร้อม spec limit line ให้เห็นภาพว่า process อยู่ใน spec หรือเปล่าทันทีค่ะ
  • 3
    ใช้ซ้ำได้กับทุก Process แค่ paste data ใหม่ ทุกอย่าง update อัตโนมัติเลยค่ะ ไม่ต้อง setup ใหม่ทุกครั้ง
Section 02

วิธีเขียน Prompt แบบ Engineer จริงๆ

สิ่งที่สำคัญกว่าไฟล์คือ วิธีคิดก่อนจะเขียน prompt ค่ะ ไม่ใช่แค่ copy-paste แล้วหวังว่ามันจะทำงานให้นะคะ

AI ต้องรู้ว่า Output คืออะไร — Excel file ที่ทำงานได้จริง ไม่ใช่แค่ตัวอย่าง

ต้องบอก Structure — column ไหนรับ input, column ไหน calculate, sheet ไหนเป็น dashboard

ต้องระบุ Edge case — เช่น ถ้า data น้อยกว่า 30 จุด ให้แสดง warning ด้วยค่ะ

📋 ตัวอย่าง Prompt Structure 
สร้าง Excel template สำหรับคำนวณ Cp และ Cpk โดย:
- Sheet 1: Data Input (คอลัมน์ A = measurements)
- Sheet 2: Calculation (USL, LSL, Mean, Std, Cp, Cpk)
- Sheet 3: Dashboard (Histogram + spec lines)

เงื่อนไข:
- รองรับ data สูงสุด 500 จุด
- ถ้า Cpk < 1.33 ให้แสดง warning สีแดง
- ใช้ซ้ำได้ แค่ paste data ใหม่

ลองปรับ prompt ให้เข้ากับ process ของตัวเองดูนะคะ ผลลัพธ์จะต่างกันเยอะมากเลยค่ะ 🎯
